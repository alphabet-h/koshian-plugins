# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-14

Initial release.

### Added

- `/nfd-init` コマンド — プロジェクトへのNFD基盤構築（ディレクトリ作成、初期ファイル生成、独立Gitリポジトリ初期化）
- `/nfd-help` コマンド — NFDワークフローの状態診断と次ステップの案内
- `crystallize` スキル — 経験データからパターンを抽出する8ステップの結晶化プロセス
- NFD方法論リファレンス（`documents/nfd-methodology.md`）
- デュアルワークスペースパターンの解説（認知コンテキスト分離の明確化）
- 過学習防止ガードと人間検証の必須化
- セッションログ、auto memory、NFDトレース、claude-mem MCP のデータソース対応
